# 📍 Consulta de CEP com API

## 📖 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma aplicação para **consulta de endereços através do CEP**, utilizando uma API pública para obter informações em tempo real.

A aplicação permite que o usuário informe um CEP válido e receba automaticamente dados como rua, bairro, cidade e estado, proporcionando uma experiência prática de integração entre Front-End e serviços externos.

---

## 🎯 Objetivos

* Consumir dados de uma API externa.
* Realizar requisições HTTP utilizando JavaScript.
* Manipular dados em formato JSON.
* Atualizar elementos da página dinamicamente.
* Desenvolver interfaces interativas.
* Aplicar conceitos de integração entre sistemas.
